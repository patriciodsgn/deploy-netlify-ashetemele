Cómo implementar una página HTML estática en Netlify

Basado en el artículo: Guía de Medium

Sobre Netlify

Netlify es un servicio de alojamiento en la nube. Ofrece hosting para aplicaciones creadas con tecnologías como:

  • Node.js
  • Ruby
  • Java
  • PHP
  • Python
  • Go
  • Scala
  • Clojure

Requisitos para Implementar en Netlify

  • Cuenta en GitHub, GitLab o Bitbucket
  • Cuenta en Netlify
  • Git instalado en tu computadora
  • CLI de Netlify instalado

Pasos para Implementar tu Página en Netlify

Netlify aloja aplicaciones, pero para implementar una página HTML estática debes seguir estos pasos:

Crear tu proyecto en Netlify:

  1. Visita Netlify Sign Up y regístrate usando GitHub, GitLab, Bitbucket o correo electrónico.
  2. Después de registrarte, selecciona "Import from Git" para importar tu proyecto existente.
  3. Conecta tu cuenta de repositorio (GitHub, GitLab, Bitbucket o Azure DevOps) y autoriza el acceso si es necesario.
  4. Elige el repositorio que deseas desplegar. Puedes otorgar acceso a todos los repositorios o seleccionar uno específico.
  5. Netlify detectará automáticamente tu configuración y comenzará el despliegue del proyecto.

Personalizar tu sitio:

  1. Dirígete a la sección de "Configuration" de tu proyecto en Netlify.
  2. En el apartado de "General", selecciona "Change site name" para personalizar el nombre del sitio si lo deseas.
  3. Tu sitio será accesible en el formato: https://[nombre_del_sitio].netlify.app/

Pasos adicionales para aplicaciones PHP:

Para páginas HTML estáticas, sigue los pasos anteriores. Si deseas alojar aplicaciones PHP, agrega los siguientes archivos:

  1. Crea un archivo index.php con el siguiente contenido: <?php include_once("home.html"); ?>
  2. Crea un archivo home.html con tu contenido HTML, por ejemplo: <h1>Hola Netlify</h1>
  3. Crea un archivo composer.json vacío con el contenido {} para que Netlify lo reconozca como una aplicación PHP.
  4. En la terminal, inicializa un repositorio Git:
    5. git init
  5. Agrega los archivos:
    6. git add .
  6. Realiza un commit:
    7. git commit -m "Desplegando"
  7. Inicia sesión en Netlify usando el CLI:
    8. netlify login
  8. Crea una nueva app en Netlify:
    9. netlify apps:create nombre_de_app
  9. Sube los cambios a Netlify:
    10. git push netlify master

Para actualizar tu proyecto, realiza los cambios, agrégalos y comitea:

git add .
git commit -m "Mensaje sobre los cambios"
git push netlify master
